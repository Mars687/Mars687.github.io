文档流

很多人或者文章称之为文档流或者普通文档流，其实标准里根本就没有这个词。如果把文档流直译为英文就是 document flow ，但标准里只有另一个词，叫做普通流（normal flow)，或者称之为常规流。但由于很多中文翻译的书就是这么来的，大家更习惯文档流的称呼。[1]

文档流是文档中可显示对象在排列时所占用的位置。比如网页的div标签它默认占用的宽度位置是一整行，p标签默认占用宽度也是一整行，因为div标签和p标签是块状对象。 网页中大部分对象默认是占用文档流，也有一些对象是不占文档流的，比如表单中隐藏域。当然我们也可以让占用文档流的元素转换成不占文档流，这就要用到CSS中属性position来控制。[2]

定位(position)

定位的整个想法是允许我们覆盖上面描述的基本文档流行为，以产生有趣的效果。

静态定位

静态定位是每个元素获取的默认值——它只是意味着“将元素放入它在文档布局流中的正常位置 。

position: static;

相对定位

它与静态定位非常相似，占据在正常的文档流中，除了你仍然可以修改它的最终位置，包括让它与页面上的其他元素重叠。可依据left，right，top，bottom等属性在正常文档流中偏移位置。

position: relative;

绝对定位

将对象从文档流中拖出，使用left，right，top，bottom等属性相对于其最接近的一个最有定位设置的父对象进行绝对定位。如果不存在这样的父对象，则依据body对象。而其层叠通过z-index属性定义 。

position: absolute;

固定定位

固定定位与绝对定位的工作方式完全相同，只有一个主要区别————绝对定位固定元素是相对于 <html> 元素或其最近的定位祖先，而固定定位固定元素则是相对于浏览器视口本身。 这意味着您可以创建固定的有用的UI项目，如持久导航菜单。

position: fixed;

布局实例

分别尝试使用Float、Position或者Flexbox来实现如下需求：

实现一个两栏布局，左侧占30%宽度，右侧占70%宽度

  • Float方法

创建了一个容器container将左右两栏放入其中，这样对于调整左右两栏整体宽度将十分方便。代码中body{margin: 0}重置浏览器默认样式中的外边距（Chrome 66.0默认外边距为8px），使网页外边距为0。这里需要注意 height属性 不可继承，所以除了将 left 和 right 设定为 height: 100% 还需将其所有父元素均设置为 height: 100%。这些父元素包括：container, body, html。只要有一个没有设置，左右两栏就无法显示出来。

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>两栏布局,左侧占30%宽度右侧占70%宽度</title>
    <style type="text/css">
        html, body {
            margin: 0;
            height: 100%;
        }
        #container{
            height: 100%;
        }
        #left, #right {
            height: 100%;    
        }
        #left {
            background: #afb;
            width: 30%;
            float: left;
        }
        #right {
            background: #0af;
            width: 70%;
            float: right;
        }
    </style>
</head>
<body>
    <div id="container">
        <div id="left"></div>
        <div id="right"></div>
    </div>
</body>
</html>

  • Flexbox方法

flex-grow 属性给予弹性项在需要的时候可以伸展的能力。它接收一个不带单位的值作为比例。它规定了在容器内的弹性项可以占用多少的可用空间。

如果所有的子项都设置了 flex-grow 为1，那么容器内的剩余空间会被均匀的分配给所有自项。如果其中一项的值为2，那么这项的占用空间会是其他项的两倍。

注 container 设定高度为100%，left 和 right 不必再设定高度。

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>两栏布局,左侧占30%宽度右侧占70%宽度</title>
    <style type="text/css">
        html, body {
            margin: 0;
            height: 100%;
        }
        #container{
            display: flex;
            height: 100%;           
        }
        #left {
            background: #afb;
            flex-grow: 3;   
        }
        #right {
            background: #0af;
            flex-grow: 7;
        }
    </style>
</head>
<body>
    <div id="container">
        <div id="left"></div>
        <div id="right"></div>
    </div>
</body>
</html>

实现一个两栏布局，左侧固定宽度(200px)，右侧根据浏览器宽度进行自适应变化

  • Float方法

1.给 container 设定 margin-left 用于放置左侧固定栏。
2.left 设定左浮动, 用负外边距将其移动到正确位置。
3.right 设定有浮动，width: 100%, 以自适应浏览器宽度变化。

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>两栏布局，左侧固定宽度(200px)，右侧自适应</title>
    <style type="text/css">
        html, body {
            margin: 0;
            height: 100%;
        }
        #container{
            height: 100%;
            margin-left: 200px;                      
        }
        #left {
            background: #afb;
            height: 100%;
            width: 200px;  
            float: left;
            margin-left: -200px;
        }
        #right {
            height: 100%;
            width: 100%;
            background: #0af;
            float: right;
        }
    </style>
</head>
<body>
    <div id="container">
        <div id="left"></div>
        <div id="right"></div>
    </div>
</body>
</html>
  • Position方法

1.给 container 设定 margin-left 用于放置左侧固定栏。
2.left 设定绝对定位将其移动到正确位置。

<!DOCTYPE html>

            height: 100%;

实现一个三栏布局，左侧固定宽度(150px)，右侧固定宽度(200px)，中间部分宽度随浏览器宽度变化而自适应变化

1.圣杯布局

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>三栏布局，左右固定宽度，中间自适应</title>
    <style type="text/css">
        html, body {
            margin: 0;
            padding: 0;
            height: 100%;
        }
        #container {
            height: 100%;
            padding-left: 150px;
            padding-right: 200px;
        }
        .column {
            height: 100%;
            float: left;  
        }
        #center {
            background: #fa0;
            width: 100%;
        }

            background: #afb;
            width: 150px;
            margin-left: -100%;
            position: relative;
            left: -150px;
        }
        #right {
            background: #0af;
            width: 200px; 
            margin-right: -200px;
        }
    </style>
</head>
<body>
    <div id="container">
        <div id="center" class="column"></div>
        <div id="left" class="column"></div>
        <div id="right" class="column"></div> 
    </div>
</body>
</html>

  • 设计稿分为头部，中间的Banner，主导航，内容区域，底部。
