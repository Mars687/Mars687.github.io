回顾一下CSS“圣杯布局”的实现：

1.先搭建布局框架，构建一个容器container，根据左右两栏的宽度给出container的内边距。

2.然后把左栏、中栏和右栏放到这个容器中。给容器中的元素添加浮动，使它们处于同一行。

此时出现了一个问题：容器的约束使左栏、中栏、右栏同时被包裹其中，由于中间栏占据了100%的可用空间，左右两栏被挤到了中间栏的下方。

3.为使左右两栏达到预定位置与中栏构成三栏布局，对左、右两栏应用负外边距和相对定位以实现目的。

“双飞翼布局”的实现

“圣杯布局”实际上是源于左右两栏固定宽度，中栏自适应的三栏布局需求。“圣杯布局”是将左、右、中三栏统一放到容器container中，通过设定container的左右内边距padding为左右两栏提供放置空间。那么我们很自然的想到：可以通过设定中间栏的外边距margin来为左右两栏提供放置空间。以下为初步源码：

<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8" />
  <title>双飞翼布局</title>

    <style>
        body {
            min-width: 550px;      /* 2x LC width + RC width */
            padding:0;
            margin:0;
            text-align: center;
            font-size: 30px;
        }
        #header {
            height: 50px;
            width: 100%;
            background-color: #fa0;
        }
        .column {
            height: 260px; 
            float: left;
        }
        #center {
            width: 100%;
            background: #D6D6D6;
            margin: 0 150px 0 200px;
        }
        #left {
            width: 200px;          /* LC width */
            background: #E6D;
        }
        #right {
            width: 150px;          /* RC width */
            background: #77BBDD;  
        }
        #footer {
            height: 50px;
            width: 100%;
            background-color: #fa0;
            clear: both;
        }
    </style>

</head>
<body>
    <div id="header">Header</div>
    <div id="center" class="column">Center</div>
    <div id="left" class="column">Left</div>
    <div id="right" class="column">Right</div>
    <div id="footer">Footer</div>
</body>
</html>


发现结果和我们预期的有出入，首先为center部分添加内边距的同时使得整个页面变宽了。实际上问题出现在

#center {width: 100%;}

当前元素宽度的比例是建立在该元素所处的容器（即父元素）基础之上的。center所在的容器是body，由于body内外边距设为0，body的宽度即浏览器页面的宽度。center部分的宽度继承自body，仍为浏览器的宽度。这里由于为column设定了左浮动，Left, Center, Right同时向左侧浮动，而中栏左侧有200px外边距，它将网页宽度向右被撑大了200px。

有一点比较奇怪，如果我们为column设定为右浮动，网页并没有被撑大（这个现象与“双飞翼布局”主题无关，仅提出一个发现），效果如下：



为解决上述问题，我们为center添加一个容器将其包裹起来，使 width：100% 和 margin: 0 150px 0 200px 两个属性分开。修改后的代码如下：

<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8" />
  <title>双飞翼布局</title>

    <style>
      body {
          min-width: 550px;      /* 2x LC width + RC width */
          padding:0;
          margin:0;
          text-align: center;
          font-size: 30px;
      }
      #header {
          height: 50px;
          width: 100%;
          background-color: #fa0;
      }
      .column {
          height: 260px; 
          float: left;   
      }
      #center {
          width: 100%;
          background: #D6D6D6;
      }
      #center-wrap {
        margin: 0 150px 0 200px;
        height: 260px;
        background: red;
      }
      #left {
          width: 200px;          /* LC width */
          background: #E6D;
      }
      #right {
          width: 150px;          /* RC width */
          background: #77BBDD; 
      }
      #footer {
          height: 50px;
          width: 100%;
          background-color: #fa0;
          clear: both;
      }
    </style>

</head>
<body>
  <div id="header">Header</div>
  <div id="center" class="column">
    <div id="center-wrap">Center</div>
  </div>
  <div id="left" class="column">Left</div>
  <div id="right" class="column">Right</div>
  <div id="footer">Footer</div>
</body>
</html>


此时实现了为左右两栏留出放置空间，通过负外边距将左右两栏放到正确的位置，代码如下：

<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8" />
  <title>双飞翼布局</title>

    <style>
      body {
          min-width: 550px;      /* 2x LC width + RC width */
          padding:0;
          margin:0;
          text-align: center;
          font-size: 30px;
      }
      #header {
          height: 50px;
          width: 100%;
          background-color: #fa0;
      }
      .column {
          height: 260px; 
          float: left;        
      }
      #center {
          width: 100%;
          background: #D6D6D6;
      }
      #center-wrap {
        margin: 0 150px 0 200px;
        height: 260px;
        background: red;
      }
      #left {
          width: 200px;          /* LC width */
          margin-left: -100%;    /* LC width */
          background: #E6D;
      }
      #right {
          width: 150px;          /* RC width */
          margin-left: -150px;   /* RC width */
          background: #77BBDD; 
      }
      #footer {
          height: 50px;
          width: 100%;
          background-color: #fa0;
          clear: both;
      }
    </style>

</head>
<body>
  <div id="header">Header</div>
  <div id="center" class="column">
    <div id="center-wrap">Center</div>
  </div>
  <div id="left" class="column">Left</div>
  <div id="right" class="column">Right</div>
  <div id="footer">Footer</div>
</body>
</html>


至此，我们得到了“双飞翼布局”。但有一点需要注意：我们对右栏设定的外边距为 margin-left: -150px。这里简单总结一下：

当元素设定为左浮动，为它添加负外边距时，

  • 若为 margin-left:
    a. margin-left < 元素宽度时，该元素会向左移动，越出其父容器。
    b. margin-left = 元素宽度时，该元素会移动到上一行块级元素最右端，仍在父容器内部，元素右边缘与父容器右边缘相重合。


  • 若为 margin-right:
    a. margin-right < 元素宽度时，该元素固定不动。
    b. margin-right = 元素宽度时，该元素会移动到上一行块级元素最右端，在父容器外部，元素左边缘与父容器右边缘相重合。


实际上，并没有从根本解释负外边距的行为逻辑，暂时没有查阅到相关资料，姑且记住结论会应用即可，日后弄清此问题再做补充。

“圣杯布局”与“双飞翼布局”异同

“圣杯布局”与“双飞翼布局”都是已知左右两栏宽度，要求中间栏自适应的布局方式。“圣杯布局”是提前设定通过包裹左、右、中三栏的容器containner的内边距来为左右两栏提供放置空间。想办法把左右两栏放到它们理应放置的位置。而“双飞翼布局”则是通过后期将中栏设定外边距，使它被左右两栏遮挡住的部分被拉回中栏该存在的位置。
其实布局思想非常简单，难点在于实现的细节处理，特别是负外边距的正确使用。
