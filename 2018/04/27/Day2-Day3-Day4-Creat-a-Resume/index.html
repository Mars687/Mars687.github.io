Day2、Day3、Day4的任务分别：

  • 创建一份简历。
  • 用CSS为它添加色彩。
  • 添加背景

创建一份简历

方法：
创建一份简历我们可以分为以下几个部分：

  • 标题
  • 基本资料
  • 专业技能
  • 项目经历

考虑到基本资料信息比较简略，可以分为两列显得较为紧凑饱满，我们采用表格的形式。而后面的专业技能与项目经历文字较多，可以采用列表的形式。
下面是初步的html代码（简历内容纯属杜撰，部分借鉴了张雯莉同学的经历[^1]）：

<!DOCTYPE html>
<html>
<head>
    <title>Resume</title>
    <link rel="stylesheet" type="text/css" href="styles/layout.css" media="screen">
</head>
<body>
    <h1>简历</h1>
    <h3>基本资料</h3>
    <table>
        <tr>
            <th>姓名：</th>
            <td>李明</td>
            <th>性别：</th>
            <td>男</td>
        </tr>
        <tr>
            <th>出生年月：</th>
            <td>1999.01</td>
            <th>民族：</th>
            <td>汉族</td>
        </tr>
        <tr>
            <th>毕业院校：</th>
            <td>IFE</td>
            <th>专业：</th>
            <td>web前端</td>
        </tr>
        <tr>
            <th>联系电话：</th>
            <td>13912345678</td>
            <th>邮箱：</th>
            <td>liming@163.com</td>
        </tr>
    </table>
    <h3>专业技能</h3>
    <ul>
        <li>熟悉html, css, javascript, Ajax, JQuery工具库</li>
        <li>熟悉node.js（会使用gulp、webpack）</li>
        <li>了解React前端框架</li>
        <li>对HTTP，TCP，UDP等网络协议有一定了解</li>
    </ul>
    <h3>项目经历</h3>
    <ol id="project">
        <li>Polyvia</li>
        <p>Low-poly 风格视频、图像处理。</p>
        <li>jWebAudio</li>
        <p>为网页游戏定制的 Web Audio 库，支持纯 JavaScript 和 jQuery 版本。</p>
        <li>Three.js 电子书</li>
        <p>《Three.js 入门指南》旨在帮助初学者学习Three.js，在图灵社区和豆瓣阅读均可免费下载。</p>
        <li>Chrome 插件</li>
        <p>划词搜索的汉典插件和 ReadFree 豆瓣传送门是两个对中文读者非常有帮助的插件。</p>
    </ol>
    <h3>实习经历</h3>
    <ul id="intern">
        <li>Intel 软件工程师</li>
        <p>2012年9月 - 2013年2月，开发了 jWebAudio、三维打砖块游戏等</p>
        <li>百度前端工程师</li>
        <p>2015年10月 - 2016年3月ECharts 热力图和 gallery</p>
        <li>阿里巴巴无线事业部前端工程师</li>
        <p>2015年7月 - 2015年9月</p>
    </ul>
</body>
</html>

Day2-制作在线简历

然后，我们为它添加些样式：

h1{
  font-size: 32px;
  text-align: center;
}

  font-size: 24px;
}

  font-weight: normal;
}

  color: #3C57C4;

  font-weight: bold;
}

  font-weight: bold;
}

至此，我们初步完成了一份简历的制作。

Day3-添加一些色彩

为了进一步美化简历，我们可以给简历添加一些背景。

h1{
  font-size: 32px;
  text-align: center;
  background: #338FCC;
}
h3{
  font-size: 24px;
  background: #338FCC;
}
p{
  color: #000;
  font-size: 16px;
  font-weight: normal;
}

  font-size: 20px;
  font-weight: bold;
}
